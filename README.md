# API

### 실시간 주가 가져오기
네이버 증권에서 크롤링하여 실시간 주가 제공

GET /predict/MACD
```json
res
{
  "time" : "(time)",
  "price" : "(number)"
}
```

### 요청 종목의 현재 추천여부

GET /predict/MACD
```json
res
```
